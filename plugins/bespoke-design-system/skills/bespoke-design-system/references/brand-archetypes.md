# Brand Archetypes — 12 原型分类系统

> 基于 Carol Pearson 与 Margaret Mark 的 *The Hero and the Outlaw*（2001），把品牌（和产品调性）归为 12 个原型。本 skill 用作画像的离散分类辅助系统。

---

## 12 原型 + 设计倾向

### 1. The Sage（智者）

- **核心动机**：理解世界，传达真理
- **典型品牌**：Google、Wikipedia、TED、Notion、Linear
- **调性词**：trustworthy / structured / clear / authoritative / calm
- **设计倾向**：
  - 中性蓝灰色调
  - 衬线 + 现代无衬线（Sans）的克制混搭
  - 严格栅格、高对比度
  - 极少装饰，信息密度高
- **避免**：花哨、情绪化、过度装饰

### 2. The Magician（魔法师）

- **核心动机**：让神奇的事发生，转化体验
- **典型品牌**：Apple、Disney、Tesla、Mailchimp（带玩味时）
- **调性词**：mystical / transformative / inspiring / visionary
- **设计倾向**：
  - 暗色调 / 深紫 / 渐变
  - 大量留白塑造"留白即魔法"
  - 慢动效（cubic-bezier 戏剧性 landing）
  - 衬线显示字体或定制字体
- **避免**：过度技术感、机械、冰冷

### 3. The Hero（英雄）

- **核心动机**：通过勇气和能力证明价值
- **典型品牌**：Nike、BMW、FedEx、Stripe
- **调性词**：confident / bold / ambitious / determined
- **设计倾向**：
  - 高饱和单色 + 黑白
  - 粗 stroke、大字号差
  - 直角或 minimal 圆角
  - 强 CTA、对称构图
- **避免**：柔弱、过度复杂、不确定

### 4. The Outlaw（叛逆者）

- **核心动机**：颠覆规则，破坏秩序
- **典型品牌**：Harley-Davidson、Diesel、Vimeo（早期）、Brave
- **调性词**：rebellious / daring / raw / countercultural
- **设计倾向**：
  - 黑红 / 高对比 / 荧光强调色
  - 直角、不规则布局
  - 粗 stroke、宽字距、大写
  - 故意"不优雅"
- **避免**：精致、企业感、对称完美

### 5. The Explorer（探索者）

- **核心动机**：寻找自由，体验新事物
- **典型品牌**：The North Face、Patagonia、Jeep、Airbnb
- **调性词**:adventurous / authentic / rugged / grounded
- **设计倾向**：
  - 自然色（土棕、森林绿、海蓝）
  - 衬线或 humanist sans
  - 中等留白、不对称布局
  - 摄影风格图像（非插画）
- **避免**：人工、过度修饰、合成色

### 6. The Creator（创造者）

- **核心动机**：创造持久的、有价值的东西
- **典型品牌**：LEGO、Adobe、Figma、Vercel
- **调性词**：imaginative / artistic / precise / creative
- **设计倾向**：
  - 多色 + 一个高饱和强调色
  - 几何形状、明确层级
  - 中等 grid 灵活性
  - 标志性微动效
- **避免**：保守、单调、标准化

### 7. The Innocent（纯真者）

- **核心动机**：体验快乐和纯粹
- **典型品牌**：Coca-Cola、Dove、Aveeno、儿童产品
- **调性词**：optimistic / pure / simple / kind
- **设计倾向**：
  - 柔和柔色调（粉、浅蓝、米白）
  - 圆角、圆形
  - humanist sans 或圆体
  - 充足留白、温和阴影
- **避免**：尖锐、冷硬、复杂

### 8. The Lover（情人）

- **核心动机**：建立亲密连接，表达激情
- **典型品牌**：Chanel、Tiffany、Godiva、Hallmark
- **调性词**：sensual / intimate / passionate / refined
- **设计倾向**：
  - 暖色中性（米、玫粉、暗金）
  - 衬线（特别是华丽衬线如 Didone）
  - 大留白 + 精致细节
  - 柔阴影、慢动效
- **避免**：粗糙、机械、冷硬

### 9. The Caregiver（照护者）

- **核心动机**：保护和帮助他人
- **典型品牌**：Johnson & Johnson、UNICEF、Volvo、医疗 SaaS
- **调性词**：nurturing / caring / reliable / safe
- **设计倾向**：
  - 蓝绿色 + 暖中性
  - 圆角、柔阴影
  - 圆体或 humanist sans
  - 高对比保证可读性、明确层级
- **避免**：冷硬、过度装饰、不确定

### 10. The Ruler（统治者）

- **核心动机**：建立秩序、控制权威
- **典型品牌**：Rolex、Mercedes、Microsoft（企业线）、IBM
- **调性词**：authoritative / prestigious / refined / structured
- **设计倾向**：
  - 深色 + 暗金强调
  - 衬线、严格栅格
  - 大字号差、对称构图
  - 极少动效、精致细节
- **避免**：轻浮、混乱、平民感

### 11. The Jester（弄臣）

- **核心动机**：享受当下，让别人开心
- **典型品牌**：Old Spice、M&M's、Mailchimp（成熟期）
- **调性词**：playful / witty / surprising / lighthearted
- **设计倾向**：
  - 多色、渐变、强对比
  - 不规则布局、手写体或 humanist
  - 大圆角、弹性动效
  - 插画 / 卡通元素
- **避免**：严肃、企业感、单调

### 12. The Everyman（凡人）

- **核心动机**：归属感、亲和力
- **典型品牌**：IKEA、Target、Levi's、Slack
- **调性词**：approachable / friendly / honest / practical
- **设计倾向**：
  - 中性 + 一个友好强调色
  - humanist sans
  - 中等圆角、轻阴影
  - 实用主义、清晰层级
- **避免**：精英感、神秘、过度高级

## 双原型组合（最常见）

实际产品很少是单一原型，常见组合：

| 组合 | 典型 | 设计调性 |
|---|---|---|
| Sage + Creator | Figma、Linear | 结构化 + 创造力，蓝灰 + 强调色 |
| Sage + Magician | Stripe（高级感时）、Apple Developer | 严谨 + 神秘，深色基调 |
| Hero + Sage | Stripe（CTA 时）、AWS | 大胆 + 可信，高对比 + 严格栅格 |
| Magician + Lover | 高端化妆品、奢侈品 | 神秘 + 亲密，暗金 + 衬线 |
| Caregiver + Sage | 医疗 SaaS、家庭管理 | 照护 + 可信，蓝绿 + 圆角 |
| Outlaw + Creator | Vimeo、Are.na | 反叛 + 创造，黑红 + 不规则 |
| Explorer + Caregiver | Patagonia、户外品牌 | 探索 + 照护，自然色 + 友好字体 |
| Ruler + Sage | IBM、Mercedes | 权威 + 智识，深色衬线 + 严格栅格 |

## 在画像中如何使用

```yaml
brand_archetypes:
  primary: Sage           # 主导调性
  secondary: Magician     # 调适方向，让产物不只是另一个 Sage
```

**原则**：

- primary 决定 60% 的设计倾向（B2 检索的 archetype 硬筛主依据）
- secondary 提供调适方向（B4 在保持 primary 主调的前提下，往 secondary 偏移）
- 同性质双原型（如 Hero + Outlaw 都是高能量）会让产物极端化，慎用
- 互补双原型（Sage + Magician 一冷一热、一显一藏）通常产生更有趣的设计

## 反原型（用于 reverse_constraint）

某些产品需要明确"不要像 X"。常见反原型表达：

- "不要像企业 SaaS"  → reverse: Ruler-corporate-coldness
- "不要像淘宝算命"  → reverse: Magician-cheap-mystical
- "不要像儿童产品"  → reverse: Innocent-saccharine
- "不要像政府网站"  → reverse: Ruler-bureaucratic

把这些反原型转成对应的 Kansei `reverse_constraint` 词。
